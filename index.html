HTML & CSS Cheatsheet

HTML

HTML Element Syntax

Opening Tag (start tag): Every element has a name which is included in the opening tag.

Attribute and Value: Attributes are different options that can be set for an element, every attribute has a value. Attributes can be used for different purposes such as classifying elements or providing link urls.

Children (content): The content of an element can either be text that will be displayed, or other elements.

Closing Tag (end tag): Every element must have a closing tag, which has a forward slash before the name of the element.

Basic Markup of a HTML page

Commonly Used Elements

Headings <h1>, <h2>,..., <h6> There are 6 heading elements going from h1 to h6, with h1 being the largest heading and h6 the smallest.
Multimedia <img/>, <video>, <audio> Tags for adding multimedia elements to a page, img is a self closing tag but audio and video require a closing tag. Attribute src defines the content of the elements.
Lists <ul>,<ol>,<li> Use these tags for creating unordered lists, ordered (numbered) lists, and list elements.
Tables <table>,<th>,<tr>,<td> Create tables, add heading elements, rows and data to the table.
Separators <hr>,<br> Both are self closing tags used to separate elements with a horizontal line, or whitespace.
Sectioning <header>, <main>, <footer>, <section>, <article>, <div>, <nav>, <figure> Elements for creating different sections of a page.
Other <a>, <p>, <link>, <title>, <b>, <figcaption>

CSS

CSS Syntax

Selector (p): the element that the CSS should be applied to

Property and Value (declaration): the styling that should be applied to the element.

Where to write CSS

External CSS is the preferred method. You have a separate file containing CSS statements and provide a link to it in the head element of the HTML document.



CSS Selectors

Basic Selectors:

elementname, .classname, #idname, [atrr=value]*

Combinators:

selectorA + selectorB, selectorA ~ selectorB, parent>child, parent descendant

Pseudo Selectors:

:active, :visited, :hover, :focus

CSS Properties

TEXT: color, font, font-family, font-size, font-weight, letter-spacing, line-height, text-align, text-decoration, text-indent, text-transform, vertical-align

LIST: list-style, list-style-image, list-style-position, list-style-type

BACKGROUND: background, background-attachment, background-color, background-image, background-position, background-repeat

DISPLAY: display, float, clear, overflow, visibility

BOX: border, border-color, border-style, border-width, height, margin, padding, width, box-sizing

POSITION: position, top, bottom, left, right, z-index
